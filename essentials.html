01

Official Foundations

Before you write a single line of code, you need to register your bot with Discord.

  • Discord Developer Portal: The "command center" where you create your application, get your bot token, and enable Intents (permissions like reading messages or member lists).
  • Discord API Docs: The holy grail. It's technical, but it's the final word on what the Discord API can and cannot do.
02

Coding Libraries (Wrappers)

Don't reinvent the wheel. Use a library that handles the heavy lifting of talking to Discord's servers.

  • JavaScript: Discord.js is the industry standard. It's fast, powerful, and has a massive community.
  • Python: Discord.py (and its popular forks like Pycord or Disnake) is incredibly beginner-friendly and readable.
  • Java: JDA (Java Discord API) is the go-to if you prefer a more structured, object-oriented approach.
03

Top-Tier Free Learning

  • YouTube: Look for channels like Tech with Tim (Python), CodeWithHarry (AI integration), or The Coding Train (JavaScript).
  • FreeCodeCamp: Full-length courses on Node.js and Python specifically tailored to building and hosting Discord bots.
  • Discord.js Guide: If you choose JavaScript, this is the unofficial manual. It walks you through every step from "Hello World" to complex databases.
04

Essential Tools & IDEs

  • Visual Studio Code (VS Code): The undisputed king of code editors. Install the "Discord" and "Prettier" extensions to make your life easier.
  • Postman: Great for testing API endpoints if you're building a bot that pulls data from external sites (like weather or game stats).
  • GitHub: Essential for version control. Plus, many free hosting services require you to have your code in a GitHub repository.
05

Free Hosting (24/7 Uptime)

Hosting is where "free" gets tricky. You don't want your bot to go offline the moment you close your laptop.

  • Oracle Cloud Free Tier: Arguably the best free VPS in 2026. It gives you a powerful "Always Free" instance that can run multiple bots easily.
  • AWS Free Tier: Offers an EC2 t2.micro instance for 12 months. Excellent for learning, but it expires after a year.
  • Phoenix Dash / SneakyHub: Community-driven free hosts that are popular for hobbyist bots.
Pro-Tip Stay away from "free" hosts that require you to "click a button every hour" to keep the bot alive. They are more trouble than they're worth.
