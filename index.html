The essentials to Discord
bot creation

Follow these steps to go from zero to a working, deployed Discord bot.

01

Create a Discord Application

Head to the Discord Developer Portal, create a new application, and generate a bot token under the Bot tab. Enable the intents your bot needs (Message Content, Server Members, etc).

02

Set Up Your Project

Install Node.js (v18+) and run npm init -y in a fresh folder. Install discord.js with npm install discord.js and store your token in a .env file.

03

Write Your First Bot

Create an index.js, instantiate a Client, listen for the ready event, and log in with your token. See the example below.

04

Register Slash Commands

Use the REST API to register commands globally or per-guild. Define a name, description, and options, then handle interactions in your interactionCreate listener.

05

Invite & Test

Use the OAuth2 URL Generator in the Developer Portal to create an invite link with bot + applications.commands scopes and add the bot to a test server.

06

Deploy

Push your code to GitHub and connect it to a host like Railway or Fly.io. Set your env variables in the platform dashboard and your bot stays online 24/7.
